# Digital Pet AI

A 2026 hackathon project: an interactive digital pet companion that learns from your voice and chat, reacts in real time, and evolves through play.

## What it does

- Responds to voice and text input
- Lets users feed, play, dance, pet, and follow the pet
- Displays mood, energy, and hunger states
- Uses AI-driven conversation for a more emotional, engaging experience

## Run locally

1. Install dependencies: `npm install`
2. Set your OpenAI API key in a `.env` file
3. Start the server: `npm start`
4. Open the app in your browser and interact with your digital pet
